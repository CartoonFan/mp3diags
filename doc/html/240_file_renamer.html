User's guide - File renamer
dfwerfwegf
The third MP3 Diags component is a file renamer / copier. It creates a new copy of a given MP3 file, based on the ID3V2 field values (artist, album, track number ...) and optionally removes the original file. doc, after file renaming: can be used to have some meaningful rating (set rating => rename => ... )
